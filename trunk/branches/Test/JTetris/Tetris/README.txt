
================== README ==================
Tetris4j is a free, open source Tetris clone
written in Java. It uses the GNU Public Lic-
ence.

Requirements to play:
 -Java 1.5 or later
 
Controls:
 Right/Left: Arrow keys
 Rotate: UP or SPACE
 Drop: Space
 Pause: P
 
Created and maintained by Baisoft.
http://code.google.com/p/tetris4j
============================================
